' #var <nombre>:<tipo>

' Tipos básicos:
' - int
' - real/float (sinónimos)
' - byte/char (sinónimos)
' - n * <tipo> (array de n elementos)
'
' En la versión avanzada del tutorial
' aparecerán el resto de los tipos 
' (estructuras, punteros y tipos de 
' usuario)
